Update to TYPO3Flow (master) - 18.10.2012

Just updated the project to the current master that is called TYPO3 Flow now instead of FLOW3.

Recent changes:

  • Update to the current version TYPO3 Flow
  • Changed to the new naming structure to be able to use composer
  • Add flattr support. Login, go to your profile and add your flattr id and now the people can flattr your code snippets.

Update to FLOW3 1.0 - 05.11.2011

I really haven't worked on this project for a while. But there have been good reasons. I follow FLOW3 as close as possible and after I noticed that there are big changes upcoming, I decided to stop working on the project until the release of beta 1. But the beta 1 of FLOW3 was released some time ago and the framework is getting better and better. So it was time to get started again and to push twitcode.org to the next level.

Recent changes:

  • Update to the current version of FLOW3
  • Added a comment function so that users can comment on each snippet. A login via Twitter is required to use the comment function.
  • Added a profil area, where you can access all your posted snippets, all your comments you ever added and where you can change your notification settings.
  • Many, many improvements, but this improvments are FLOW3 related and are not visible to you. But perhaps you feel the improved performance coming along with the FLOW3 1.0 version.
